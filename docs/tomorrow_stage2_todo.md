- [ ] Chia template thành các block rõ ràng:
  - [ ] imports
  - [ ] repo path setup
  - [ ] target import
  - [ ] fixed oracle contracts
  - [ ] `Driver` class
  - [ ] approach-specific methods
  - [ ] `TestOneInput`
  - [ ] `_SEED_CORPUS`

- [ ] Chuẩn hóa comment `# LLM_FILL` để mỗi slot có format giống nhau.
- [ ] Đảm bảo mỗi slot nói rõ:
  - [ ] LLM được phép sửa gì
  - [ ] return value cần là gì
  - [ ] không được động vào phần nào
- [ ] Kiểm tra lại 3 strategy branch:
  - [ ] `recorded_call`
  - [ ] `return_value`
  - [ ] `filesystem_state`
- [ ] Với `recorded_call`, đảm bảo skeleton đã cố định phần oracle check ngoài `try/except`.
- [ ] Với `return_value`, đảm bảo skeleton chỉ assert trên extracted return value.
- [ ] Với `filesystem_state`, quyết định có giữ strategy hiện tại không, vì snapshot chỉ trong `allowed_root` chưa đủ để detect outside-root write.

```bash

```
